# Extra cool information (helpful reference)

Optional material that is **not** used by the scripts in `scripts/`. Keep grids, shortcuts, and older chart exports here so the repo root stays clean.

- **PNG files:** Earlier chart exports (before outputs were standardized under `images/`).
- **Excel (`grid1_*.xlsx`):** Your grid / analysis workbooks.
- **`.lnk` files:** Windows shortcuts to related files; not read by Python.

The pipeline still reads only from `data/` and writes to `images/`.
